# MESSENGER APP in ANGULAR & NESTJS

## Features

1. User Authentication & Registration 
2. Typing Indicators
3. New Message Events

## Tech used
1. Angular for client
2. Nestjs for Backend
3. Typeorm and MYSQL/XAMP
4. Passport Js for Authentication
